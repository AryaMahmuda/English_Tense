1. Simple Present Tense

Digunakan untuk menyatakan fakta, kebiasaan, atau kejadian yang terjadi berulang-ulang saat ini.

Rumus Active

[+] S + V1(s/es) + O/C
[-] S + do/does + not + V1 + O/C
[?] Do/Does + S + V1 + O/C?

Contoh Kalimat Active

KalimatPenjelasan
(+)He reads a book every day.Sebuah kebiasaan yang dilakukan rutin.
(-)He does not read a book every day.Menyatakan negasi dari kebiasaan.
(?)Does he read a book every day?Menanyakan tentang kebiasaan tersebut.

Rumus Passive

[+] S + is/am/are + V3 + by + O
[-] S + is/am/are + not + V3 + by + O
[?] Is/Am/Are + S + V3 + by + O?

Contoh Kalimat Passive

KalimatPenjelasan
(+)A book is read by him every day.Fokus pada objek (buku) yang dikenai tindakan.
(-)A book is not read by him every day.Menyangkal bahwa buku itu dibaca olehnya.
(?)Is a book read by him every day?Menanyakan apakah tindakan itu dilakukan pada buku.

2. Simple Past Tense

Digunakan untuk menyatakan suatu kejadian yang terjadi dan selesai di masa lampau.

Rumus Active

[+] S + V2 + O/C
[-] S + did + not + V1 + O/C
[?] Did + S + V1 + O/C?

Contoh Kalimat Active

KalimatPenjelasan
(+)She wrote a letter yesterday.Aksi menulis sudah selesai di masa lampau.
(-)She did not write a letter yesterday.Menyangkal kejadian di masa lampau.
(?)Did she write a letter yesterday?Bertanya tentang kejadian di masa lampau.

Rumus Passive

[+] S + was/were + V3 + by + O
[-] S + was/were + not + V3 + by + O
[?] Was/Were + S + V3 + by + O?

Contoh Kalimat Passive

KalimatPenjelasan
(+)A letter was written by her yesterday.Fokus pada objek (surat) yang ditulis.
(-)A letter was not written by her yesterday.Menyangkal bahwa surat itu ditulis olehnya.
(?)Was a letter written by her yesterday?Menanyakan apakah surat itu ditulis olehnya.

3. Simple Future Tense

Digunakan untuk menyatakan suatu kejadian yang akan terjadi di masa depan.

Rumus Active

[+] S + will + V1 + O/C
[-] S + will + not + V1 + O/C
[?] Will + S + V1 + O/C?

Contoh Kalimat Active

KalimatPenjelasan
(+)They will build a new house.Sebuah rencana atau prediksi di masa depan.
(-)They will not build a new house.Menyatakan rencana itu tidak akan terjadi.
(?)Will they build a new house?Menanyakan tentang rencana di masa depan.

Rumus Passive

[+] S + will be + V3 + by + O
[-] S + will not be + V3 + by + O
[?] Will + S + be + V3 + by + O?

Contoh Kalimat Passive

KalimatPenjelasan
(+)A new house will be built by them.Fokus pada objek (rumah) yang akan dibangun.
(-)A new house will not be built by them.Menyangkal bahwa rumah itu akan dibangun.
(?)Will a new house be built by them?Menanyakan tentang rencana pembangunan rumah.

4. Present Continuous Tense

Digunakan untuk menyatakan kejadian yang sedang berlangsung saat ini atau sekitar waktu pembicaraan.

Rumus Active

[+] S + is/am/are + V-ing + O/C
[-] S + is/am/are + not + V-ing + O/C
[?] Is/Am/Are + S + V-ing + O/C?

Contoh Kalimat Active

KalimatPenjelasan
(+)I am studying English now.Aksi sedang terjadi pada saat berbicara.
(-)I am not studying English now.Menyangkal aksi yang sedang terjadi.
(?)Are you studying English now?Menanyakan tentang aksi yang sedang terjadi.

Rumus Passive

[+] S + is/am/are + being + V3 + by + O
[-] S + is/am/are + not + being + V3 + by + O
[?] Is/Am/Are + S + being + V3 + by + O?

Contoh Kalimat Passive

KalimatPenjelasan
(+)English is being studied by me now.Fokus pada objek (English) yang sedang dipelajari.
(-)English is not being studied by me now.Menyangkal bahwa English sedang dipelajari.
(?)Is English being studied by you now?Menanyakan tentang aksi yang terjadi pada objek.

5. Present Perfect Tense

Digunakan untuk menyatakan kejadian yang sudah terjadi di waktu yang tidak spesifik di masa lampau tapi masih ada hubungannya dengan sekarang, atau baru saja selesai.

Rumus Active

[+] S + have/has + V3 + O/C
[-] S + have/has + not + V3 + O/C
[?] Have/Has + S + V3 + O/C?

Contoh Kalimat Active

KalimatPenjelasan
(+)She has finished her homework.Aksi sudah selesai dan hasilnya (PR selesai) terlihat sekarang.
(-)She has not finished her homework.Menyatakan aksinya belum selesai hingga sekarang.
(?)Has she finished her homework?Menanyakan status penyelesaian tugasnya.

Rumus Passive

[+] S + have/has + been + V3 + by + O
[-] S + have/has + not + been + V3 + by + O
[?] Have/Has + S + been + V3 + by + O?

Contoh Kalimat Passive

KalimatPenjelasan
(+)Her homework has been finished by her.Fokus pada objek (PR) yang telah diselesaikan.
(-)Her homework has not been finished by her.Menyatakan bahwa PR tersebut belum diselesaikan.
(?)Has her homework been finished by her?Menanyakan apakah PR-nya sudah diselesaikan.
